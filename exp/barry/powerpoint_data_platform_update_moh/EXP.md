# PowerPoint Data Platform Update MoH EXP

Status: candidate handover

This EXP captures how Barry wants Ministry of Health / NSW Health style data
platform project update decks created and updated.

The source material is a real project update deck. The goal is to learn the
deck structure, update rhythm, visual restraint, and stakeholder communication
style so future agents can produce similar decks faster and with higher
quality.

## Current Learned Shape

The April 2025 training deck is a monthly project update presentation with:

- cover slide using NSW Health / Audit Office identity
- section divider slides
- monthly update label and period
- key developments slide with dated milestones
- project timeline overview split across two slides
- data pipeline slide with visual diagram
- key risks and forward plan table
- related business section
- Q&A / thank-you slide
- upcoming meetings slide
- closing/back slide

## Style Direction

Use a restrained government/stakeholder presentation style:

- clean white or light background
- strong but limited use of brand colour
- large clear headings
- concise executive phrasing
- timeline/status information presented as structured blocks
- risk information presented in tables
- avoid decorative stock imagery or generic business graphics

## Training Status

The source deck has been saved, structurally extracted, and rebuilt through a
blank/template-context training run. The accepted rebuild is v9, which visually
matches the 13-page source deck within a strict slide-image comparison
threshold.

## Current PowerPoint Creation Rule

For this EXP, "from scratch" should mean:

1. start a new output deck using the learned source deck as template context
2. preserve master/layout/theme and provided visual assets
3. clear and rebuild editable text using PowerPoint rich-text clear/paste
4. retain complex visual/table assets when rebuilding them changes rendering
5. export to PDF/PNG and compare slide-by-slide before handover

Pure white-slide reconstruction is useful for learning coordinates, but it is
not the production route for exact MoH project update decks.
