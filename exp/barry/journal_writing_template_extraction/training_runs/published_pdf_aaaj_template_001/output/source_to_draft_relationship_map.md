# Source PDF to Barry Draft Relationship Map

## Core relationship

The published AAAJ PDF is the source structure. Barry's draft is an adaptation
of that structure to the NFIA/audit society topic.

```text
Published article PDF
  -> extract writing structure
  -> generate fillable SLR writing template
  -> apply template to Barry's NFIA topic
  -> Barry's draft
```

## Macro-structure mapping

| Published PDF structure | Barry draft adaptation | Structural function |
|---|---|---|
| Structured abstract | Structured abstract | Compress purpose, method, findings, implications, originality |
| Introduction | Introduction | Establish field shift, gap, RQs, contribution, roadmap |
| Antecedents/theoretical background | Conceptual Foundations | Build conceptual lens before review method |
| Review design and exploratory analyses | Review Design and Methodology | Justify corpus, search, screening, and analytical approach |
| Narrative review and discussion of contributions | Analysis and Thematic Synthesis | Turn reviewed literature into themes and contribution logic |
| Discussion and conclusions | Discussion and Conclusion | Return to RQs, theory, and implications |
| References | References | Journal-style scholarly apparatus |
| Appendix | Optional appendix/supporting materials | Transparency and supplementary detail |

## What the future EXP should learn

The future EXP should not learn the published article's content. It should
learn the reusable writing moves:

- how the introduction stages the problem
- how the theory/background section prepares the review
- how methodology is justified as systematic and transparent
- how tables/figures make the review credible
- how the synthesis moves from literature description to conceptual claims
- how the discussion returns to theory and contribution

## What to ignore

- exact topic claims
- exact findings
- article-specific citations
- author-specific wording
- page layout unless the user asks for visual formatting

## Validation idea

A good extracted template should help produce a structurally similar paper on a
different topic without copying the source article's claims or wording.
