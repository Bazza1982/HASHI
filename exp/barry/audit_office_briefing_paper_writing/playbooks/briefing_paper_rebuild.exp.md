# Briefing Paper Rebuild Playbook

Use this playbook when training against a sample Audit Office briefing paper.

## Goal

Rebuild a new Word document that matches the sample paper's structure, fonts,
colour, table layout, and executive briefing rhythm without treating the source
topic as reusable content.

## Workflow

1. Copy the source sample into the EXP training material folder.
2. Export the source to PDF and render pages for visual evidence.
3. Extract DOCX structure:
   - paragraph sequence
   - style usage
   - run fonts and colours
   - tables
   - media count
4. Build a role map:
   - front briefing metadata
   - purpose
   - proposal/rationale
   - feedback
   - in-scope
   - out-of-scope
   - strategic alignment
   - governance/team
5. Rebuild a practice document from blank or template context.
6. Export rebuilt document to PDF.
7. Validate with:
   - page count
   - paragraph/style parity
   - table count and table layout
   - visual page comparison
   - text placeholder/content checks

## Important Constraint

When the goal is exact visual reproduction, Word styles and table rendering may
require template context. Pure blank-page reconstruction is useful for learning,
but final-quality rebuilds should preserve the source style context and rebuild
content deliberately.
